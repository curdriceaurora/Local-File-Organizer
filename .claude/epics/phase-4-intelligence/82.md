# Issue #82: Rename format parameter to avoid shadowing

## Description

The ImageMetadata.__init__() method uses format as a parameter name, which shadows Python's built-in format() function.

## Current Problem

- Built-in Shadowing: Makes format() function inaccessible within scope
- Can confuse developers and static analysis tools
- Considered poor practice by style guides

## Proposed Solution

Rename parameter to image_format or fmt:
- Keep self.format attribute name (external API)
- Only change parameter name
- Update all call sites
