# Issue #79: Replace deprecated IOError with OSError

## Description

The text extractor uses the deprecated IOError exception type instead of the modern OSError.

## Current Problem

- Deprecated: IOError is an alias for OSError since Python 3.3
- Ruff Warning: Tool flags this as UP024 violation
- Best Practice: Modern Python code should use OSError

## Proposed Solution

Update both docstring and exception type:
- Change docstring: "IOError" → "OSError"
- Change exception: raise IOError(...) → raise OSError(...)
